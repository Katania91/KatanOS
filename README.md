# KatanOS

KatanOS is a local-first desktop productivity app built with Electron, React, and TypeScript.

It combines everyday tools in a single application:

- dashboard
- agenda and events
- todo and checklists
- finance tracking
- contacts
- habits
- journal
- bookshelf
- encrypted vault
- mini-games

## Project Status

This repository is published as an open baseline of the project.

The original planned scope is complete, and the codebase is now available for anyone who wants to fork it, extend it, or adapt it.

## Quick Start

```bash
npm ci
npm run electron:dev
```

Build desktop package:

```bash
npm run electron:build
```
